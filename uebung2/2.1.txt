2.1a
Variablen haben eine endliche Länge. Daher kan ein loop-Befehl durch aneinanderreihen von endlich vielen Ps
dargestellt werden. Da die restlichen Befehle auch endliche Länge besitzen kann somit das gesammte Programm mit endlich vielen Befehlen die jeweils endlich viel Zeit benötigen ausgefüht werden. 
D.h. es terminiert in endlicher Zeit

2.1b

x:=1
WHILE x != 0 DO y:=1 END

2.1c
LOOP x1 DO P END 

x2:= x1 + 0
WHILE x2 !=0 DO P; x2:x2-1 END

2.1d
(i)
result:=y+0; x1:=x+0;
WHILE x1 != 0 DO
	x1:=x1-1;

(ii)
